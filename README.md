# Freight Matching Agent

> Agentic AI prototype for continuous freight-searching and logistics decision support.

## Overview

This project explores how an AI Agent can support continuous freight-searching workflows in logistics scenarios. The prototype focuses on task decomposition, freight-matching logic, tool-assisted reasoning, and result validation.

Competition result: **Top 24 / 963, finalist award**.  
Official competition page: https://tianchi.aliyun.com/competition/entrance/532467/information

## Problem

Continuous freight-searching involves repeated decision-making under changing supply-demand information. A useful agent should help users:

- Understand current freight-searching objectives.
- Retrieve or reason over candidate freight information.
- Compare candidate options.
- Explain trade-offs.
- Support human-in-the-loop decisions.

This public repository is a desensitized project write-up. It will avoid private data, restricted competition materials, and any non-public business details.

## Agent Workflow

```text
User request
  -> intent recognition
  -> task decomposition
  -> tool / data query
  -> candidate generation
  -> rule-based or model-assisted evaluation
  -> result explanation
  -> human confirmation
```
