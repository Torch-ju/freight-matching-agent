# 满帮动态决策智能体简历可粘贴版

## 项目经历

**阿里天池 AI 算法大赛：满帮连续找货智能体**  
**Agentic AI 物流在线决策系统｜独立完成｜2026.05 - 2026.06**

- **项目背景：** 针对货运司机连续找货中收益、时间窗、空驶成本和个性化偏好难以统一决策的问题，设计在线 Agent 决策系统，基于官方接口可见的司机状态、候选货源、历史动作和自然语言偏好，连续输出 `take_order / wait / reposition` 动作，完成月度经营优化闭环。
- **偏好编译与规则转化：** 围绕“偏好文本 -> 可执行约束与评分函数 -> 运筹算法优化”的主链路，构建自然语言偏好到可执行规则的转化流程，将时间窗、地点任务、数量目标、跨月补偿、数量上限、禁忌规则等归纳为通用偏好类型，并通过字段规范化、关键条件覆盖检查和规则试运行降低 LLM 解析漂移。
- **在线运筹决策：** 针对单票收益贪心无法处理长期机会的问题，设计 ServiceNode / ServicePlan 滚动短视界规划框架，统一比较真实货源、等待、空驶和偏好虚拟任务，综合毛收入、距离成本、目标缺口、长途上限和时间窗风险，提升有限时间内的序贯接单质量。
- **工程闭环：** 搭建本地仿真、参数实验、冻结包、边界扫描和全量测试流程，自动记录毛利、成本、净利、偏好罚分、Token 和动作日志；本地公开评测最高净利达到 `114856.46`，并通过平台隐藏集反馈定位隐藏偏好编译泛化不足的问题，沉淀完整复盘文档。
